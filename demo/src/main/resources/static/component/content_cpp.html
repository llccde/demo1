第 1 章：C++ 简介

C++ 是一种面向对象的编程语言，由贝尔实验室的 Bjarne Stroustrup 于 1979 年开发。C++ 语言在 C 语言的基础上增加了面向对象的特性。

第 2 章：C++ 基础语法

C++ 的基本语法与 C 语言相似，程序由函数组成，最基本的函数是 main() 函数。

int main() {
    // 程序入口
    return 0;
}

第 3 章：数据类型

C++ 支持多种数据类型，包括基本数据类型和用户自定义数据类型。

  • 基本数据类型：int, char, float, double
  • 用户自定义数据类型：struct, class, enum

第 4 章：控制结构

控制结构用于控制程序的执行流程，包括条件语句和循环语句。

条件语句

if (condition) {
    // 执行代码
} else {
    // 执行其他代码
}

循环语句

for (int i = 0; i < 10; i++) {
    // 执行代码
}

第 5 章：函数

函数是执行特定任务的代码块，可以提高代码的重用性。

int add(int a, int b) {
    return a + b;
}

第 6 章：类与对象

C++ 是一种面向对象的编程语言，支持类和对象的概念。

class Dog {
public:
    void bark() {
        std::cout << "Woof!" << std::endl;
    }
};

int main() {
    Dog myDog;
    myDog.bark();
    return 0;
}

第 7 章：异常处理

C++ 提供了异常处理机制来处理运行时错误。

try {
    // 可能抛出异常的代码
} catch (const std::exception &e) {
    // 处理异常
}

第 8 章：标准模板库 (STL)

STL 是 C++ 的一个强大库，提供了许多常用的数据结构和算法。

#include <vector>

std::vector numbers = {1, 2, 3, 4, 5};

第 9 章：文件操作

C++ 提供了文件输入输出的功能，可以方便地读写文件。

#include <fstream>

std::ofstream outFile("output.txt");
outFile << "Hello, World!";
outFile.close();

第 10 章：总结

本教程介绍了 C++ 的基础知识和常用特性。希望读者能够通过本教程掌握 C++ 编程。
